# 春招冲鸭

## 数组与字符串

### 3. 无重复字符的最长子串

分类：**滑动窗口**

如果右边加入一个字符产生重复，左边吐出一个字符继续判断。

```c++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int i=0, j=0, maxsize = 0;
        map<char, bool> m;
        while(j<s.size()){
            char c = s[j];
            if(!m[c]){
                m[c] = true;
                j++;
            }else{
                maxsize = max(maxsize, j-i);
                m.erase(s[i]);
                i++;
            }
	    }
	
	    maxsize = max(maxsize, j-i);;
	    return maxsize;
    }
};
```

```c++
class Solution {
public:
	// 道理是一样的，写法不同而已
    int lengthOfLongestSubstring(string s) {
        int i=0, j=0, maxsize = 0;
        map<char, bool> m;
        for(int i=0; i<s.size(); i++){
            while(j<s.size() && !m[s[j]]){
                m[s[j]] = true;
                j++;
            }
            maxsize = max(maxsize, j-i);
            m.erase(s[i]);
        }
	    return maxsize;
    }
};
```
